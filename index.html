Video 6 - tsconfig y json

El archivo tsconfig es un archivo en formato json que se crea en el directorio raiz de mi proyecto typescript. Esto se hace para indicarle al proyecto typescript las diferentes opciones del compilado.

Por ejemplo le puedo indicar que archivos deseo compilar y cuales excluir

En un proyecto con una estructura simple, una carpeta llamada app, un archivo app.ts y un index.html se puede ejemplificar.
